# Backend-Experiment9: Pagination

## Steps

### 1. Use Existing Database
Use the `backend_experiments` database and `posts` table from previous experiments.

### 2. Place Files
Put all files in:
MAMP: /Applications/MAMP/htdocs/experiment9  

Files:
- db_connect.php
- pagination.php

### 3. Run
MAMP:  
http://localhost:8888/experiment9/pagination.php  

### 4. Navigation
Use "Previous" and "Next" links or page numbers at the bottom to navigate between pages. Each page shows 5 posts.
